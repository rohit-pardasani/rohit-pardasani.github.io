Bias-Variance Tradeoff¶

By Rohit Pardasani¶

Suppose that we have a training set consisting of a set of points $x_{1},x_{2}.....,x_{n}$ and real values $y_{i}$ associated with each point $x_{i}$.
We assume that there is a function with noise $y=f(x)+\epsilon$, where the noise, has zero mean and variance $\sigma ^{2}$.

What is this meaning of this expecation $E_D ?$
This is expectation over multiple training datasets. We know we have just one dataset D. But there could be infinitely many datasets that could be generated from the underlying function say $D_1,D_2,D_3,D_4,......$. If we choose a different training set we will arrive at a different estimator, $\hat{f}(x)$. So, $E_{D}$ means we are taking expectation over possible training sets.
