What is Linear Regression ?

1. Supervised machine learning algrorithm for performing regression task.

2. Linear Regression approach is based on assumption that there is a linear relationship between dependent variable (Y) and independent variables (X s).

Simple Linear Regression

Simple linear regression focuses on determining relationship between one independent (explanatory variable) variable and one dependent variable.

Linear Regression Model

$\hat{y} = mx + c$

$m$ = Slope
$c$ = Intercept
$\hat{y}$ = Predicted value of y

The aim is to minimize cost function $J(m,c)$


$J(m,c) = \frac{1}{2N} \sum_{i=1}^{N} (\hat{y}_i-y_i)^2$

$J(m,c) = \frac{1}{2N} \sum_{i=1}^{N} (mx_i+c-y_i)^2$

Using partial derivative $m$ and $c$ can be calculated

Basically, we are minimizing Mean Square Error (MSE)


$MSE = \frac{1}{N} \sum_{i=1}^{N} (\hat{y}_i-y_i)^2$

How good is our Linear Regression fit ?


1. Mean Square Error (MSE = $\frac{1}{N} \sum_{i=1}^{N} (\hat{y}_i-y_i)^2$) : Lower the MSE better is the fit.
Interesting Result: If we have minimized mean square error then sum of residuals is zero i.e. $\sum_{i=1}^{N} (\hat{y}_i-y_i) = 0$

2. Coefficient of Determination (R2 or r2) :
Is defined as proportion of the variance in the dependent variable that is predictable from the independent variables.

Total sum of squares = SStot = $\sum_{i=1}^{N} (y_i - \bar{y})^2$ : Proportional to the variance of data

Sum of squares of residuals = SSres = $\sum_{i=1}^{N} (y_i - \hat{y}_i)^2$ : Proportional to the mean square error

$R^2 \equiv 1 - \frac{SS_{res}}{SS_{tot}}$

In the best case, the modeled values exactly match the observed values, which results in SSres = 0 and R2 = 1.
A baseline model which always predicts $\bar{y}$ will have R2 = 0.
Models that have worse predictions than this baseline will have a negative R2.
