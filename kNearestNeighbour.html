k Nearest Neighbour (kNN) with Python¶

By Rohit Pardasani¶

k-nearest neighbors algorithm (k-NN) is a non-parametric classification method


In k-NN classification, the output is a class membership. An object is classified by a plurality vote of its neighbors, with the object being assigned to the class most common among its k nearest neighbors (k is a positive integer, typically small).
If k = 1, then the object is simply assigned to the class of that single nearest neighbor. In k-NN regression, the output is the property value for the object. This value is the average of the values of k nearest neighbors.

Example of k-NN classification. The test sample (green dot) should be classified either to blue squares or to red triangles.
If k = 3 (solid line circle) it is assigned to the red triangles because there are 2 triangles and only 1 square inside the inner circle.
If k = 5 (dashed line circle) it is assigned to the blue squares (3 squares vs. 2 triangles inside the outer circle).

The best choice of k depends upon the data; generally, larger values of k reduces effect of the noise on the classification, but make boundaries between classes less distinct.
A good k can be selected by various heuristic techniques.
The special case where the class is predicted to be the class of the closest training sample (i.e. when k = 1) is called the nearest neighbor algorithm.

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
from matplotlib.lines import Line2D
plt.figure(figsize=(6, 6))
legend_elements = [Line2D([0], [0], marker='o', color='#1f77b4', label='Class 1'),
                   Line2D([0], [0], marker='o', color='#ff7f0e', label='Class 2'),
                   Line2D([0], [0], marker='x', color='black', label='Unknown')]
np.random.seed(7)
colordict = {'Class 1':'#1f77b4','Class 2':'#ff7f0e'}
x1 = np.random.normal(loc=0.0, scale=1.0, size=15)
y1 = np.random.normal(loc=4.0, scale=2.0, size=15)
x2 = np.random.normal(loc=5.0, scale=2.0, size=20)
y2 = np.random.normal(loc=0.0, scale=1.0, size=20)
df = pd.DataFrame(columns=['X','Y','Class'],dtype=float)
df['X'] = list(x1)+list(x2)
df['Y'] = list(y1) + list(y2)
df['Class'] = ['Class 1']*15 + ['Class 2']*20
display(df)
cols = [colordict[k] for k in df['Class'].values]
plt.scatter(df['X'].values,df['Y'].values,color=cols,s=65)
x3 = np.random.normal(loc=2.5, scale=3.0, size=3)
y3 = np.random.normal(loc=2.0, scale=1.0, size=3)
plt.scatter(x3,y3,marker='x',c='black',s=35)
plt.legend(handles=legend_elements)
plt.show()

from sklearn.neighbors import KNeighborsClassifier
labelDict = {'Class 1':0,'Class 2':1}
labelDictRev = {0:'Class 1',1:'Class 2'}
D = list(zip(df['X'].values,df['Y'].values))
L = [labelDict[k] for k in df['Class'].values]
neigh = KNeighborsClassifier(n_neighbors=3)
neigh.fit(D, L)
Dt = list(zip(x3,y3))
Dtp = neigh.predict(Dt)
Dtpc = [labelDictRev[k] for k in Dtp]
x = list(df['X'].values) + list(x3)
y = list(df['Y'].values) + list(y3)
classes = list(df['Class'].values) + list(Dtpc)
cols = [colordict[k] for k in classes]
plt.figure(figsize=(6, 6))
plt.scatter(x,y,color=cols,s=65)
plt.scatter(x3,y3,color='black',marker='x',s=35)
plt.legend(handles=legend_elements)
